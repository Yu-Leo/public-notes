Идея:

PublicNotes - место, где каждый может оставить заметку на общей стене.

Функциональность:

  • Заметки:
    • Отображение одним списком (главная страница)
    • Отображение по категориям
    • Поиск по названиям
    • Добавление
    • Редактирование
    • Удаление
  • Отображение списка авторов
  • Профиль:
    • Регистрация
    • Отображение профиля и заметок
    • Изменение данных профиля
    • Смена пароля
    • Удаление профиля
  • Категории:
    • Отображение списка
    • Добавление

Версии:

  • Добавлен поиск по названиям заметок
  • К категориям добавлены картинки
  • На странице категории добавлена кнопка "Добавить заметку", позволяющая создать заметку сразу в данной категории
  • Исправлен баг с отсутствием меню навигации на мобильных устройствах
  • Исправлен баг с меню навигации на странице "О проекте"

Используемые технологии:

  • BackEnd:

      • Django (3.1.14)
      • django_debug_toolbar (2.2.1)
    • Веб-сервер: gunicorn
  • FrontEnd:
    • Языки: html, css
    • Фреймворки и библиотеки:
      • Bootstrap 5
  • Система контроля версий: git
  • IDE: JetBrains PyCharm Community
