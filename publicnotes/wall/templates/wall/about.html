Идея:

PublicNotes - место, где каждый может оставить заметку на общей стене.

Функциональность:

  • Отображение заметок одним списком (главная страница)
  • Отображение заметок по категориям
  • Отображение списка авторов
  • Аккаунт:
    • Регистрация
    • Отображение профиля
    • Изменение данных профиля
    • Добавление заметок

Используемые технологии:

  • BackEnd:
    • Язык программирования: Python (3.10)
    • Фреймворки и библиотеки:
      • Django (3.0.2)
      • django_debug_toolbar (2.2)
  • FrontEnd:
    • Языки: html, css
    • Фреймворки и библиотеки:
      • Bootstrap 5
  • Система контроля версий: git
  • IDE: JetBrains PyCharm Community
