Идея:

PublicNotes - место, где каждый может оставить заметку на общей стене.

Функциональность:

  • Заметки:
    • Отображение одним списком (главная страница)
    • Отображение по категориям. Каждая заметка может принадлежать одной категории
    • Отображение по тегам. Каждая заметка может иметь несколько тегов
    • Счётчик кол-ва просмотров
    • Поиск по названиям
    • Добавление публичных и приватных заметок
    • Возможность закреплять заметки (закреплённые заметки будут отображаться сверху списка на странице профиля)
    • Рейтинговая система: каждый лайк повышает рейтинг на 1, каждый дизлайк - уменьшает
    • Редактирование
    • Удаление
  • Отображение списка авторов
  • Профиль:
    • Регистрация
    • Подтверждение e-mail по одноразовой ссылке
    • Отображение профиля и заметок
    • Рейтинговая система: рейтинг пользователя - сумма рейтингов его публичных заметок
    • Изменение данных профиля
    • Смена пароля
    • Удаление
  • Категории:
    • Отображение категорий в виде списка с различными уровнями вложенности
    • Каждая категория может иметь подкатегории. Заметки могут относиться как к родительской, так и к дочерним категориям
  • Поддержка русского и английского языков

Версии:

  • Исправлены ошибки
  • Общее:
    • Добавлена поддержка английского языка в интерфейсе
    • Добавлены тесты
    • Добавлена возможность запускать сервер в Docker с использованием Docker compose
    • Переведена и улучшена документация
    • Оптимизированы запросы к БД
    • СУБД сменена на PostgreSQL
  • Заметки:
    • Добавлен поиск по названиям заметок
    • Добавлена возможность создавать приватные заметки
    • Добавлено отслеживание кол-ва просмотров у заметок
    • Отредактированные заметки помечаются фразой "изменено" возле даты создания
    • Добавлена возможность закреплять заметки в профиле
    • Добавлена рейтинговая система для заметок
    • На странице заметки добавлена кнопка "Редактировать" для авторов заметки
  • Категории и теги:
    • К категориям добавлены картинки
    • На странице категории добавлена кнопка "Добавить заметку", позволяющая создать заметку сразу в данной категории
    • Добавлена возможность создавать подкатегории в каждой категории
    • Убрана возможность создавать категории пользователям. Категории может создавать только админ
    • Добавлены теги. Каждая заметка может содержать несколько тегов. Добавлена возможность просматривать статьи по тегам
    • В меню пагинации добавлены ссылки на первую и на последнюю страницы
  • Пользователи:
    • Добавлена функциональность отправки электронных писем для подтверждения e-mail после регистрации
    • Добавлена рейтинговая система для профилей
